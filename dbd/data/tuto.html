Tutorial

Tab "Configure Perks"

Step1 : Select the side ("Survivor" or "Killer") ... or use the dedicated button to randomly select it :)

Step2 : Set the weight of each perk using various ways

  • Click the "set default weights" button to set predefined weights for each perk (loaded data at start)
  • Click the "set identical weights" button to set the same weight for each perk
  • Click the "open custom perk distribution" button to load custom weights for each perk
  • Manual edit of weights directly from the table

Step3 : Don't forget to save the current perk distribution ... if you manually modified weights from the table


Custom weights will be loaded at start if configuration file ("perks_db_custom.txt") is found in the directory

Tab "Generate Random Builds"

Step1 : Set the number of perks in each build

Step2 : Set the number of wanted random builds

Step3 : Enable/disable constraints about care/sprint perks in generated builds

Step4 : Click the "get random builds" button to generate and display random builds

The perks from the 1st random build are graphically displayed in the table
